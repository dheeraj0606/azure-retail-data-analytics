# Retail Analytics Platform (Azure)

End-to-end Azure Data Engineering project demonstrating modern enterprise data platform architecture.

## Tech Stack

- Azure SQL Database
- Azure Synapse Analytics (Spark + Pipelines)
- Azure Data Lake Storage Gen2
- Azure Key Vault
- Azure DevOps CI/CD
- Microsoft Fabric (OneLake + Semantic Models)
- Power BI

## Architecture

OLTP → Synapse Spark → Delta Lake → Semantic Model → Power BI

## Infrastructure

Provisioned using Bicep and PowerShell automation.
